# Frequently Asked Questions

## General

**Q: What is AegisPCAP?**
A: An AI-driven network security intelligence platform for threat detection and analysis.

**Q: Is it open source?**
A: Yes, licensed under MIT License.

**Q: What file formats are supported?**
A: PCAP, PCAPNG formats are supported.

## Technical

**Q: What ML models are used?**
A: Ensemble of Isolation Forest, One-Class SVM, and LOF models.

**Q: Can I add custom detectors?**
A: Yes, through the plugin system. See [Plugin Development](../developer-guide/plugin-development.md).

**Q: How do I scale for large networks?**
A: Use Kubernetes deployment with horizontal pod autoscaling.

## Community

**Q: How do I contribute?**
A: See [Contributing Guide](../../CONTRIBUTING.md).

**Q: Where can I get help?**
A: GitHub Issues, Discussions, or community forum.

**Q: Can I share my trained models?**
A: Yes, through the Model Registry.
